# Депозит учащегося / Донат на ОРС

> **Частота:** 25 сообщений · 24 диалогов из 3 064

## Описание
Ученик просит оставить средства вместо возврата. **24 диалога**.

## Типичные фразы учеников
- «Предлагаю не возвращать, пусть останется в фонде»
- «Оставьте как донат на развитие ОРС»
- «День добрый! Прошу депозит на курс по Ведической литературе использовать»
- «Деньги сейчас более актуальны, чем депозит»

## Варианты
- **Депозит учащегося** — зачтётся при следующей записи на любой курс
- **Донат на развитие ОРС** — безвозмездный вклад

## Шаблон ответа (депозит)
```
🍀 Добрый день! Переносим ваш депозит [X] руб. на курс «[Курс]».
[Если частичное покрытие]: Депозит покроет [N] блока из [M].
При следующей записи — сообщите нам 🙏
```

## Шаблон ответа (донат)
```
💖 Благодарим за Вашу поддержку Общества!
Средства [X] руб. зачислены как донат на развитие ОРС 🙏
```
